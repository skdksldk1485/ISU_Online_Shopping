
#### 사용기술
- HTML5 / CSS3
- JavaScript(ES6+)
- React
- Redux, Redux-thunk
- MongoDB, Mongoose
- Node.js
- Express
- SCSS
- Gsap

- 페이지네이션
- 상품 검색

- 상품 리뷰
- 주문 Checkout 프로세스

- 페이팔/신용카드 결제기능
- 주문확인용 사용자 마이 페이지 , 프로필
